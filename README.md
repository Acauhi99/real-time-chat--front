# Real Time Chat

A modern real-time chat application built with React and TypeScript.

## Technologies 🚀

- React
- TypeScript
- Material UI 6
- Vite
- RxJS
- React Router
- Axios
- Zod for validation

## Installation 📦

```bash
git clone https://github.com/username/real-time-chat--front.git
cd real-time-chat--front
npm install
```
